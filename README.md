### 🔭 What I'm building

**ZENITH** — an embeddable hybrid search engine, written from scratch in Go. Custom LSM storage engine (WAL → MemTable → SSTable → leveled compaction), inverted posting lists, BM25 lexical ranking fused with ONNX-based semantic retrieval via weighted Reciprocal Rank Fusion, served over gRPC with Prometheus instrumentation and a Cobra CLI. Improved Recall@10 from 0.812 → 0.906 on MS MARCO (100k passages) by fixing tokenizer padding, batching ONNX inference, and correcting a vocabulary warm-up cache bug.
